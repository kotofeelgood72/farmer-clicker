## 4. ТЕХНИЧЕСКИЕ ТРЕБОВАНИЯ

| Параметр | Значение |
|---|---|
| Формат музыки | `.ogg` (Vorbis) или `.mp3`, 128–192 kbps |
| Формат SFX | `.ogg` или `.wav`, 44.1 kHz, mono |
| Общий вес аудио | до 10–15 МБ (Яндекс Игры) |
| Loop | бесшовный, fade в начале/конце ≤ 50 ms |
| Громкость | нормализовать до -14 LUFS |
| Библиотека воспроизведения | Howler.js |

---

## 5. РЕКОМЕНДАЦИИ ПО ИНТЕГРАЦИИ

- **Музыка:** один активный трек, crossfade 1–2 сек при смене сцены.
- **SFX:** ограничить одновременное воспроизведение клика по наковальне (1 раз в 50–80 мс), чтобы не было «каши».
- **Громкость:** музыка ~50%, SFX ~80% по умолчанию, раздельные слайдеры в настройках.
- **Mobile:** все аудио lazy-load после первого пользовательского жеста.
- **Кеширование:** хранить в `src/assets/audio/` с раздельными папками `music/` и `sfx/`.
